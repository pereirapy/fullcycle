## Playground
    git clone git@github.com:pereirapy/fullcycle3.git

## Quasar + Node + Postgres
    - cd quasar
    - docker-compose up
    - if you want to use db aws, you have to create a .env at quasar/be/.env and put the following vars:
        - NODE_ENV=production
        - PG_HOST
        - PG_USERNAME
        - PG_DATABASE
        - PG_PASSWORD
